Download

Latest release: Version 1.1.0 for Eclipse 3.0 and later

Download the plugin and unzip it into your plugins directory. The zip file contains a directory called com.teaminabox.eclipse.subtext_x.y.z, which when placed under the Eclipse plugins directory will add the SubText plugin to your Eclipse installation.
Note that you need to restart Eclipse to make the plugin available.

Open the SubText view from Window menu / Show View / Other ... / Subtext.

SubText Packs

The beginnings of a set of standard packs of SubText documents is included in the plugin zip file. To use them, copy the directory subtextPacks to a project (anywhere).
